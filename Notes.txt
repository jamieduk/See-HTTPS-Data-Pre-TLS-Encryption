How to test:
Generate certificates: openssl req -new -x509 -days 365 -nodes -out cert.pem -keyout key.pem
Run the sniffer: Use the bpftrace command shown in the video (3:02) to hook SSL_write_ex.
Run the server: python3 secure_app.py
Run the client: python3 login.py
The bpftrace tool will display the plain-text user:mysecretpassword before the TLS encryption wraps it for network transmission.
